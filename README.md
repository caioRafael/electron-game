# Game Project

Projeto Node.js com TypeScript.

## Instalação

```bash
npm install
```

## Scripts Disponíveis

- `npm run build` - Compila o TypeScript para JavaScript
- `npm start` - Executa o código compilado
- `npm run dev` - Executa o código TypeScript diretamente (desenvolvimento)
- `npm run watch` - Compila e observa mudanças nos arquivos

## Estrutura do Projeto

```
game/
├── src/          # Código fonte TypeScript
├── dist/         # Código compilado JavaScript (gerado)
├── package.json
└── tsconfig.json
```
